# **App Name**: CareerPilot AI

## Core Features:

- Secure Login: Implement OAuth-based authentication using Google and GitHub, storing user data in MySQL and managing sessions with JWT tokens.
- Resume Builder: Allow users to upload or build resumes, extract text, and store versions in MySQL.
- AI Resume Analysis: Provide an AI-generated ATS score and improvement suggestions for uploaded resumes.
- Interview Question Generator: Generate dynamic, role-based interview questions using AI. Roles include DevOps, Software Engineer, Cloud Engineer and Data Analyst.
- AI Mock Interview Simulator: Allow users to answer interview questions and receive AI-based evaluations on correctness, clarity, and confidence, along with model answers and improvement suggestions.
- Skill-Gap Analyzer: Compare user resume skills against job role requirements to identify missing skills and generate personalized learning roadmaps. It will function as a tool by incorporating or not free resources into the user's roadmap.
- Job Description Analyzer: Allow users to paste a job description, extract required skills and experience, and generate a resume-JD match percentage, highlighting missing keywords.

## Style Guidelines:

- Primary color: Strong purple (#9D4EDD) for innovation.
- Background color: Very light purple (#F5EEFF) for calm neutrality.
- Accent color: Darker pink (#E94560) for focus.
- Body and headline font: 'Inter' (sans-serif) for a clean, modern user experience.
- Use a consistent set of line icons for key features, with the purple shade as a highlight color.
- Maintain a clean and intuitive layout, focusing on ease of use for students and recruiters.
- Use subtle animations for feedback and transitions to enhance user engagement.
